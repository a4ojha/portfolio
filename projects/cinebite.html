cineBite
Java Spring Boot MongoDB Node.js React Bootstrap CSS

I developed a full-stack web application that lets users add reviews for the latest top movies.

The backend is developed in Java 21.0.1 and deployed using the Spring Boot framework. MongoDB is used for the database.

The frontend is developed using React 18.2.0 and Bootstrap.

About

Want to know what movie to watch Friday night? Want unbiased and unfiltered opinions from your community? Welcome to cineBite, a user-friendly space for a plethora of communities. Whether you're a film bro or a casual viewer, cineBite will be a hub for all things movie discussion.
